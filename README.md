# Contact Form with Client-Side Validation

 🚀🚀Project Overview
This project is a simple contact form built using **HTML, CSS, and JavaScript** with client-side validation. The form ensures that:
- The **Name**, **Email**, and **Message** fields are not empty.
- The email input follows a **valid email format** using regex validation.
- Users get **error messages** for incorrect inputs.
- A **success message** appears when the form is correctly filled.

 📝Tech Stack Used
- **HTML**: Structure of the form.
- **CSS**: Styling the form and layout.
- **JavaScript**: Handling validation and user input.


 ✏️Setup Instructions
1. Download the project files and place them in a folder.
2. Make sure all files are inside the same directory.
3. Open index.html in Google Chrome or any browser.
4. If using **VS Code**, open the project and run using Live Server if there else use the googleChrome.

 ⚒️How the Form Works
✔ Users enter **Name, Email, and Message**.  
✔ JavaScript **validates** the inputs:  
   - Name and Message **cannot be empty**.  
   - Email must match a **valid email pattern** using regex.  
✔ If any input is invalid, **error messages** are displayed.  
✔ If all inputs are valid, a **success message** appears.  

 👩‍🏫Code Explanation
- **`index.html`**: Contains the form structure.
- **`styles.css`**: Styles the form, centers it, and adds a background image.
- **`script.js`**: Handles input validation using JavaScript.
